## 编译说明
本译文由新晋排版引擎 [Typst](https://typst.app) 编译。

如需本地编译，请遵[官方指导](https://github.com/typst/typst/#installation)下载命令行工具 `typst`，并克隆本仓库到本地。

进入仓库目录，执行如下命令编译：

```bash
typst c ./译文.typ
```
